# Cellular Automata
***Autor:*** Marco Antonio Garrido Rojas. 2019


### Dependencies
- cmake
- sfml
- g++

## Installation

Install sfml:

***Ubuntu:***

```bash
sudo apt-get update
sudo apt-get install libsfml-dev
```
***Arch Linux:***

```bash
sudo pacman -S sfml

```
Clone this repository and compile:

```bash
git clone https://github.com/MagnesiaReal/Cellular_Automata_0-255.git
cd Cellular_Automata_0-255
make
cd bin/
./automata_celular

```
